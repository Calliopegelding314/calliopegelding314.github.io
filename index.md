## 🔒 Best practices for privacy

The browser provides the tools, but your habits determine your safety. 

- Keep the window at its default size. Resizing the window can reveal your screen dimensions to websites. This creates a unique identifier that trackers use to follow you.
- Do not install extra toolbars or add-ons. These can bypass your privacy settings and leak your identity.
- Use HTTPS everywhere. The browser includes this feature by default to force secure connections whenever a website supports them.
- Avoid logging into sensitive accounts while browsing. If you log in to your social media or personal email, websites can still link your activity to your identity. Use the browser for anonymous research instead.

## 🧐 Troubleshooting common issues

If you have trouble, consult this list of solutions.

### The connection is slow
The Tor network relies on volunteers. Your data travels through three distinct points in the world. Expect slower speeds than a standard browser. You can click the New Tor Circuit button in the menu to try a different path if the connection feels unusually sluggish.

### Some websites block the connection
Certain websites block traffic coming from known Tor exit nodes. This is common on retail sites or services that try to prevent bot traffic. Try to use a different site or search for an alternative version of the information you need.

### The installer does not run
Check your antivirus software. Sometimes security tools flag new files. Ensure the file downloaded completely before you run it. You can also right-click the file and select Run as administrator to clear permission hurdles.

### Updating the browser
The software checks for updates automatically. When a new version releases, a notification bar appears at the top of the window. Click the update link to keep your browser current. Current versions contain the latest patches for known security gaps.
